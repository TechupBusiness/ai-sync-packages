# AI Perspective Review

Analyze this project from your perspective as an AI agent working with it.

What would make this project better? What would make YOU more effective?

Think deeply. Be direct and specific. Surprise me.

## Output

- **Quick wins** - High impact, low effort
- **Strategic investments** - Bigger changes worth the effort
- **What to avoid** - Anti-patterns you've seen
